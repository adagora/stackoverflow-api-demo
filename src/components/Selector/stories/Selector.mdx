# Selector

The `Selector` component is used to create a customizable select input.

## Usage

The `Selector` component accepts the following props:

- `label`: (string) The label for the select input.
- `value`: (string) The current selected value.
- `onChange`: (function) Callback function triggered on value change.
- `style`: (object) Custom styles for the select input.
- `error`: (boolean) Indicates if there is an error state.
- `disabled`: (boolean) Disables the select input when true.

```jsx
import React from "react";
import Selector from "./Selector";

const MyComponent = () => {
  const handleChange = (event) => {
    // Handle selection change
  };

  return (
    <div>
      <Selector
        label="Select an option"
        value="option1"
        onChange={handleChange}
        error={false}
        disabled={false}
      >
        <SelectorOption value="option1" label="Option 1" />
        <SelectorOption value="option2" label="Option 2" />
        <SelectorOption value="option3" label="Option 3" />
      </Selector>
    </div>
  );
};
```
